Review4You! - Python Flask Web App

ABOUT

Review4You! is a python based Flask microframework web application that makes use of the Yelp Fusion API and GoogleMaps API. The user searches for places based on key terms combined with their zip code, and a google map is populated with markers and InfoBoxes based on yelp's API response given your parameters.

AREAS OF FOCUS

  • Python 3.6 & Flask MicroFramework
  • Https API requests: GET & POST
  • oAuth security for API requests
  • Bootstrap 4
  • Hosted on Heroku Public Cloud

FisherLink - .NET Web App

ABOUT

FisherLink is student registration application that ultilizes ASP.NET Core and SQL server. You may create, edit, and delete database entities such as Students,Courses, Enrollments. Both GoogleMaps API and Facebook Login Authentication API were implemented.

AREAS OF FOCUS

  • ASP.NET Core & C#
  • Hosted on Azure - including SQL Server instance
  • Entity Framework & Migrations
  • Local Authentication & Authorization, plus Facebook Authentication
  • Facebook Authentication API Integration
